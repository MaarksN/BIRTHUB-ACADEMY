# White-label B2B

## Recursos

- Marca por tenant.
- Domínio customizado.
- Tema visual.
- Turmas corporativas.
- Relatórios para gestores.
- Licenças por assento.
- Certificados co-branded.
- Trilhas customizadas.
- Webhooks.

## Regra crítica

White-label nunca pode quebrar isolamento multi-tenant.
